> **這是一個結合「醫材管理」與「床旁即時紀錄」的智慧照護管理中樞。**  
> 本專案針對 **太平洋醫材 (PAHSCO)** 臨床命題開發，旨在解決護理人力短缺、行政負擔沉重與醫材盤點混亂等臨床痛點。

## 💡 設計理念 (Design Philosophy)

在醫療缺工的背景下，護理師每班花費超過 **20% 的時間** 在處理盤點、重複登錄資訊與繁瑣交班。「護理晴報」的設計核心是將醫材數位化、資訊結構化，實現「行動床旁護理」，預期能**減少 40-60% 的行政工時**。

---

## ✨ 四大核心功能模組 (Core Modules)

### 1. 🎯 當班任務與緊急提醒 (Home)
* **主動預警：** 自動推播高風險事項（如導尿管更換到期、異常生命徵象），降低臨床遺漏。
* **任務管理：** 床旁即時勾選醫囑執行狀況，大幅提升紀錄的時效性。

### 2. 📦 醫材智慧盤點 (Supplies)
* **即時扣帳：** 整合掃描功能，讓護理師在病床旁操作時即時扣除庫存，建立詳細稽核軌跡。
* **水位監控：** 自動監測安全庫存與效期，過期或低水位時主動警示，降低報廢損失。

### 3. 👥 病患動態追蹤 (Patient Profile)
* **侵入性耗材追蹤：** 以圖表式管理導尿管、引流管等留置天數，強化院感預防。
* **影像記錄：** 支援即時拍照上傳（如患部傷口記錄），建立完整的醫療證據鏈。

### 4. 🔄 AI 自動優先級交班 (Smart Handover)
* **智慧摘要：** 系統自動抓取當班數據，並將報告分類為 **「需繼續觀察（綠）」**、**「需處理（黃）」**、**「需補貨（棕）」**。
* **效率提升：** 減少交班文書負擔，確保照護資訊精準傳遞。

---

## 🛠️ 技術亮點 (Technical Highlights)

* **行動化響應式 UI：** 針對行動裝置優化介面設計，確保護理師在移動過程（護理車/手機/平板）中操作流暢。
* **視覺化回饋：** 使用 `lucide-react` 結合色標警告系統，將臨床風險資訊「視覺化」。
